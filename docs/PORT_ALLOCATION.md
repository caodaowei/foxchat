# 端口规划

本项目端口规划遵循 AITest01 统一规范，详见：

**[AITest01 端口规划文档](../openclaw/docs/PORT_ALLOCATION.md)**

## FoxChat 使用的端口

| 服务 | 端口 | 说明 |
|------|------|------|
| Backend | 7001 | API 服务 |
| Frontend | 7002 | Web 界面 |
| PostgreSQL | 7801 | 数据库 |

## 访问地址

- **前端**: http://localhost:7002
- **后端 API**: http://localhost:7001/api
- **API 文档**: http://localhost:7001/docs
