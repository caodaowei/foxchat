# FoxChat 端口规划文档

## 端口分配原则

| 范围 | 用途 | 说明 |
|------|------|------|
| 5001-5999 | 本地开发 | 开发环境使用 |
| 7001-7799 | 生产应用 | 正式部署的应用服务 |
| 7801-7999 | 数据库/基础设施 | PostgreSQL、Redis 等 |

## 端口冲突处理

如遇端口冲突，检查步骤：

```bash
# 查看端口占用
sudo netstat -tlnp | grep :7001

# 查看 Docker 容器
sudo docker ps -a

# 停止冲突服务
sudo docker stop <container_name>
```
