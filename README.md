# 狐语 FoxChat

> 团队沟通协作平台
> Team Communication & Collaboration Platform

---

## 项目名称

- **中文**: 狐语 (Huyu)
- **英文**: FoxChat
- **寓意**: 狐狸之间的语言，团队沟通的桥梁

---

## 项目定位

OpenClaw 项目交付团队的第一个实际项目，构建团队沟通协作平台。

---

## 核心功能

- 📝 **多项目管理** - 创建、管理多个项目
- 💬 **沟通记录** - 记录团队沟通过程
- 📢 **飞书通知** - 自动推送重要消息
- 👥 **成员状态** - 查看团队成员工作状态
- 📊 **统计报表** - 项目、沟通、成员统计
